# Rakesh Androtula | Professional Portfolio

A premium, modern portfolio website showcasing the professional journey, expertise, and projects of Rakesh Androtula, Senior Engineering Manager at Experian.

[Live Demo](https://rakeshandrotula.github.io/)

## üöÄ Vision & Design Philosophy

The website is designed with a **"Classic Modernist"** aesthetic, blending the timeless elegance of the **Georgia Serif** typeface with cutting-edge web technologies and **Material Design** principles.

- **Glassmorphism**: Subtle translucency and backdrop blurs create a sophisticated layered depth.
- **Dynamic Interaction**: An interactive "Red Smoke" water ripple effect on the canvas provides a unique, engaging background.
- **Micro-Animations**: Purposeful, scroll-triggered animations (reveals, scales, and slides) guide the user through the narrative.
- **Soothing Theme**: A carefully curated color palette featuring desaturated reds and deep grays for a "Premium Tech" feel.

## üíª Local Development

To run the project locally, you only need a simple HTTP server.

```bash
# Using Python
python3 -m http.server 8080

# Or using Node.js
npx serve
```

Open `http://localhost:8080` in your browser.

## üì¶ Deployment

The site is built specifically for **GitHub Pages**.

1. Push the code to the `main` branch of your repository.
2. Go to **Settings > Pages** in your GitHub repository.
3. Select "Deploy from a branch" and choose the `main` branch and `/root` folder.
